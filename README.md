### My Weather App

O My Wheather é um app para consulta de dados climáticos da localição atual do
usuário. 

O app aprensenta uma dashboard com os principais dados climáticos:

- Cidade
- Temperatura
- Umidade
- Nunvens
- Visibilidade
- Ventos
